# Cabo App - Legal & Privacy Infrastructure

Este repositorio contiene la documentación legal oficial, las políticas de privacidad y los términos de servicio para el ecosistema digital de **Cabo App**. 

La infraestructura está diseñada para garantizar el cumplimiento de las normativas internacionales de protección de datos y los requisitos de seguridad del protocolo **"Black-Card"**.

## 🌐 Enlaces Oficiales
- **Portal de Privacidad:** [https://angelo.am/politica-de-privacidad/](https://angelo.am/politica-de-privacidad/)
- **Desarrollador:** Angelo Aranda Montaño

## 🛡️ Propósito del Repositorio
Este repositorio aloja de forma pública y transparente las políticas que rigen nuestras aplicaciones móviles en la Google Play Store. Asegura que los usuarios finales de **Cabo App** tengan acceso constante a la información sobre el tratamiento de sus datos personales.

## 📋 Documentación Incluida
- **Privacy Policy:** Protocolos de recopilación de datos para servicios de lujo (Villas, Yates, Transporte).
- **Terms of Service:** Reglas de uso para Miembros VIP y Comisionistas.
- **Seguridad Black-Card:** Detalles técnicos sobre la restricción de capturas de pantalla y cifrado de transacciones.

## 🛠️ Tecnologías
- **Frontend:** HTML5 / CSS3 (Luxury Minimalist Design)
- **Seguridad:** Encriptación de extremo a extremo
- **Hosting:** GitHub Pages / Custom Domain (angelo.am)
